Welcome to SmartBlocks
If you see this page, it means your SmartBlocks installation worked !

So what have we here ?

This is the default "home" application created by the command : 

.../yourproject/$ smartblocks init [project-name]

It is hosted in the default block "main". This block's aim is to unify the features and apps of all your other blocks.

In this block, you'll put stuff like global stylesheets, the homepage, the about us page and so on. All the stuff that is really specific to the website being developped.

All right. Why create other blocks then ?

In other blocks, you'll put features and apps that can be reused for future projects. For example, if you need a "Contact Us" page, with a form that saves messages in the database, this feature can be reused in other websites, and even shared with the community.
To create such a block, say you want to call it "contact", you'll have to run the following command : 

.../yourproject/$ smartblocks create_block contact

This will create the basis structure for the block.
