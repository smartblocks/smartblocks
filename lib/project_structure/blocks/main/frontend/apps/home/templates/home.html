Welcome to SmartBlocks
If you see this page, it means your SmartBlocks installation worked !

To learn more about SmartBlocks, please read our readme on Github : here.

What can I do now ?

You can modify the home application (which is currently showing here), by editing the code in /blocks/main/frontend/apps/home/.

You can generate blocks, which are basically modules to keep your different features independent. 

$ smartblocks generate_block

You can generate types, which creates models, controllers and everything to persist data : 

$ smartblocks generate_type

You can generate apps, which creates views, templates and stylesheets to create front-end apps. 

$ smartblocks generate_type
